# AbootC-
AbootC is an Android bootloader engine designed to understand how bootloader locking and unlocking works. It will include built-in tools like fastboot, recovery, and EDL. AbootC will also feature its own AOSP builder (go to recovery to start the build this system is only available on AbootC)

AbootC is an Android bootloader engine designed to help understand how bootloader locking and unlocking works. It includes built-in tools like fastboot, recovery, and EDL. AbootC also features its own AOSP builder - go to recovery to start the build. This system is only available on AbootC. AbootC will include multiple built-in libraries for working with it, and AbootC itself is written in C and assembly.
